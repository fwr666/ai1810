day04 笔记
　　字符串的格式化表达式
　　　　作用：
　　　生成一定格式的字符串
　运算符　%
 语法格式：
 　　格式化字符串 % 参数值
 　　格式化字符串 % (参数值1,参数值2，...)
 示例：
 　 fmt = "name: %s, age:%d"
    n = 'tarena'
    a = 15
    result = fmt % (n,a) #result = 'name : tatena, age : 35'

格式化字符串中的占位符和类型码
　　占位符和类型码　　　意义
　　　　%s           字符串（str）函数转换
       %r            字符串，使用repr(x)函数转换
       %c           整数转换为单个字符
       %d            十进制整数
       %o             八进制整数
       %x             十六进制整数(字符a-f小写)
       %X                十六进制整数(字符a~f大写)
       %e              指数型浮点数(e小写)
       %E              指数型浮点数（E大写）
       %f,%F            十进制浮点数
       %g,%G            十进制形式浮点数指数浮点自动转换
       %%                等同于一个%字符


占位符和类型码之间的格式化语法：
　　% [- + 0 宽度.精度]类型码
　说明：
　　- 左对齐（默认为右对齐）
   + 显示正号
   0 左侧空白位置补零    只能在左边加，不能加在右边
   宽度　整数,整个数据输出的宽度(占用的字符数)
   精度　整数,保留小数占后多少位(默认为6位)

循环语句：
　　循环语句：
　　　　while 语句
       for   语句
    问题：
    　　输入一个整数,写程序打印如下：
    　　　　　　这是第1行
    　　　　　　这是第2行
              ....
              这是第n行

作用：　
　　while：根据一定条件，重复执行一条或多条语句
  语法格式：
  　　whlie 真值表达式：
  　　　　语句块１（此部分可能重复执行）
    else: 
      语句块2

    说明：
    　　１.先执行真值表达式，测试布尔值为True或False
        2.如果真值表达式测试布尔值为True,则执行语句块1,然后再返回到第一步,
        　重复进行测试
        3.如果真值表达式为False,则执行语句块2,然后结果此while语句的执行
      　　,如果没有else语句,则直接结束此while语句的执行
        注：　else字句部分可以省略(同if 语句类似)

    break 语句
    　问题：
    　　如果在循环过程中，不想再继承执行此循环了，怎么办？
      作用：
      　　用于循环语句（while或for语句中）,用来终止当前循环语句的执行
      语法:
        break
     说明：
     　　当break语句执行后，此循环break之后的语句将不再执行
        break 语句通常和if 语句组合使用
        break 语句终止循环,循环语句的else字句将不会执行
        break 语句只能终止当前循环语句的执行，如果有循环嵌套时，不会
        　　　　跳出外面的循环
        break 语句只能在循环语句(while for 循环中)

        死循环　death loop
           死循环是指循环条件一直成立的循环
           死循环通常用break语句来终止循环
           死循环的else 子句永远不会执行
